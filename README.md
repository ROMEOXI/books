# BookList

![Image 1](image/image1.png)

> Make a booklist! This is a React Application.

Features:

- Create the book
- Edit the book
- Delete the book

### Install packages

```bash
npm i
```

### In two separate terminal windows, run the following commands:

Terminal Window 1:

```bash
npm start
```

Terminal Window 2:

```bash
npm run server
```
